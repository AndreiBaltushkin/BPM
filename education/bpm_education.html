AJAX helps accomplish making requests to load/send information. AJAX allows interacting with a website's server.

API exist to store data (plain data to cut out HTML, CSS, and Javascript junk) that servers might request (interaction between softwares). Web API are HTTP-based so that servers can use them for websites. Requests are made to API's URL.

XML used to be the standard format for API-s, but nowadays it is replaced by JSON format.

This website contains the documentation for JSON format. Generally, it was developed to suit well for Javascript.

This website helps with json syntax.

Postman and Hopp Scotch (free) allow testing requests by URL.

Request status codes provide information about request (e.g. 200 = OK, 404 = Not Found)
